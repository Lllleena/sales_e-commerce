# Анализ данных по продажам: EDA и RFM-сегментация

## Описание проекта

Этот проект представляет собой полный анализ датасета по продажам. Включает в себя:

- **Exploratory Data Analysis (EDA)**: выявление ключевых паттернов, проблем с данными (возвраты, outliers), визуализации (барчарты, boxplot, heatmap корреляций).
- **Очистка данных**: удаление отрицательных значений, пропусков, дубликатов и outliers.
- **RFM-анализ**: сегментация клиентов по Recency (давность покупки), Frequency (частота) и Monetary (денежная ценность).
- **Рекомендации**: на основе результатов — оптимизация бизнеса (снижение возвратов, удержание клиентов).

Проект реализован в Jupyter Notebook (`sales.ipynb`) на Python. Данные обрабатываются с учетом кодировки ISO-8859-1. 

Цель: Помочь бизнесу понять структуру продаж, выявить риски и возможности для роста.

## Структура проекта

- `sales.ipynb`: Основной Jupyter Notebook с кодом, визуализациями и выводами.
- `data.csv`: Датасет по продажам.
- `README.md`: Описание проекта.

## Данные

Датасет `data.csv` содержит столбцы:
- InvoiceNo: Номер покупки.
- StockCode: Артикул товара.
- Description: Описание товара.
- Quantity: Количество.
- InvoiceDate: Дата покупки.
- UnitPrice: Цена за единицу.
- CustomerID: ID клиента.
- Country: Страна.

Обработка: Преобразование дат, расчет TotalPrice (Quantity * UnitPrice), фильтры по годам/месяцам.

Проблемы в данных: Отрицательные значения (возвраты), пропуски в CustomerID, outliers.

## Ключевые результаты

### EDA
- **Возвраты**: Доминируют товары 23843 и 23166 (>70% от volume возвратов). Рекомендация: Проверить качество/логистику этих позиций.
- **Распределения**: После очистки — средний чек ~13.7£, фокус на объемных продажах (Quantity 1-10 в 75% случаев).
- **Визуализации**: Гистограммы топ-товаров, boxplot по ценам/количеству, heatmap корреляций.

### RFM-сегментация
- Клиенты разделены на сегменты: Champions, Loyal Customers, Potential Loyalists, At Risk, Hibernating и др.
- ~50% — лояльные клиенты (высокие F и M).
- ~20% — группы риска (низкий Recency).
- Визуализация: Polar chart с нормализованными RFM-профилями сегментов.

### Рекомендации
- **Снижение возвратов**: Фокус на топ-возвратных товарах; улучшить описания/фото.
- **Удержание клиентов**: Кампании для At Risk (скидки, напоминания); персонализация для Potential.
- **Оптимизация ассортимента**: Усилить продажи топ-товаров; мониторить outliers для оптовых сделок.
